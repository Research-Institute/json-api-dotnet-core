JSON:API .NET Core

A json:api complaint web application framework for .NET Core

Read more Go to the docs

Objectives

The goal of this package is to facilitate the development of APIs that leverage the full range of features provided by the json:api specification.

Eliminate boilerplate

Typical CRUD boilerplate code is eliminated and a wide range of features across your resources endpoints are provided.

Extensibility

This library relies heavily on an open-generic-based dependency injection model, which allows for easy per-resource customization.

Features

The following features are supported, from HTTP all the way down to the database

Filtering

Perform complex filtering using the filter parameter with our flexible client-side query syntax

Sorting

Order resources on one or multiple attributes using the sort parameter

Pagination

Leverage the benefits of paginated resources with the page parameter

Sparse fieldset selection

Get just the data that you need using the fields parameter

Relationship inclusion

Side-load related resources of nested relationships using the include parameter

Extensibility

Override and extend a broad range of elements of the framework to customize it for your own needs
